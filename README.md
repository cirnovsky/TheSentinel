# Run The Sentinel locally

This repo runs locally with Vite and uses OpenAI credentials for Codex/OpenAI API access.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create `.env.local` (or `.env`) and set:
   `OPENAI_API_KEY=your_openai_api_key`
   `OPENAI_MODEL=codex-latest` (optional)
3. Run the app:
   `npm run dev`
